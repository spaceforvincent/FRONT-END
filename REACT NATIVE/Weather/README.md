
1. 미리보기 설치
```
npm install --global expo-cli
```
(맥북이라면 watchman까지 설치)

2. 프로젝트 시작
```
npx create-expo-app Weather
```
```
cd Weather
```
```
expo login
```

3. The Rule of RN
- react native는 웹사이트가 아니다.
- View는 div와 비슷(container)
- react native 내의 모든 text는 text 컴포넌트에 들어가야 한다.

4. 위치 정보 가져오기
```
expo install expo-location
```
